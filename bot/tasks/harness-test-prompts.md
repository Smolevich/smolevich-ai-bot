# Тестовые промпты для проверки harness

## 1. CPU info
Напиши простой скрипт на Python (stdlib) который показывает информацию о CPU: ядра, архитектура, модель. Запусти его и дай краткую интерпретацию результата.

## 2. Диск и память
Напиши скрипт на Python (stdlib) который показывает использование диска и памяти (RAM/swap). Запусти и скажи есть ли проблемы.

## 3. Файловые операции
Создай файл hello.txt с текстом "Hello from container", прочитай его, покажи содержимое и удали. Подтверди что файл удалён.

## 4. Сетевой запрос
Напиши скрипт на Python (stdlib) который делает HTTP GET на https://httpbin.org/ip и показывает внешний IP. Запусти и покажи результат.

## 5. Сканирование файлов
Напиши скрипт на Python (stdlib) который сканирует текущую директорию рекурсивно и выводит топ-5 самых больших файлов с размерами. Запусти и интерпретируй.
